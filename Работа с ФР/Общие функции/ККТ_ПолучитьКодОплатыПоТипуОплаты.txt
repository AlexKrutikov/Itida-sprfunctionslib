IF ( ( ПУСТО( ТИПОПЛАТЫ ) AND ТИПОПЛАТЫ <> 0 ) OR ПУСТО( МОДЕЛЬОБОРУДОВАНИЯ ) )
{
	СООБЩЕНИЕ("Не удалось определить код оплаты в ККТ по типу оплаты и модели оборудования", ФР.ИМЯОБОРУДОВАНИЯ() );
	RETURN false;
}

IF ( МОДЕЛЬОБОРУДОВАНИЯ == "АТОЛДТО10" )
{
	IF ( ТИПОПЛАТЫ == 0 )
		RETURN "cash";
	IF ( ТИПОПЛАТЫ == 1)
		RETURN "electronically";
	IF ( ТИПОПЛАТЫ == 2 )
		RETURN "prepaid";
	IF ( ТИПОПЛАТЫ == 3 )
		RETURN "credit";
	IF ( ТИПОПЛАТЫ == 4 )
		RETURN "other";
}
ELSE IF ( МОДЕЛЬОБОРУДОВАНИЯ == "МЕРКУРИЙ" )
{
	IF ( ТИПОПЛАТЫ == 0 )
		RETURN "cash";
	IF ( ТИПОПЛАТЫ == 1 )
		RETURN "ecash";
	IF ( ТИПОПЛАТЫ == 2 )
		RETURN "prepayment";
	IF ( ТИПОПЛАТЫ == 3 )
		RETURN "credit";
	IF ( ТИПОПЛАТЫ == 4 )
		RETURN "consideration";
}
ELSE IF ( МОДЕЛЬОБОРУДОВАНИЯ == "ШТРИХ" )
{
	IF ( ТИПОПЛАТЫ == 0 )
		RETURN 1;
	IF ( ТИПОПЛАТЫ == 1 )
		RETURN 2;
	IF ( ТИПОПЛАТЫ == 2 )
		RETURN 14;
	IF ( ТИПОПЛАТЫ == 3 )
		RETURN 15;
	IF ( ТИПОПЛАТЫ == 4 )
		RETURN 16;
}

//если дошли досюда, значит не определили код оплаты
СООБЩЕНИЕ("Не удалось определить код оплаты в ККТ по типу оплаты " + ТИПОПЛАТЫ + " и модели оборудования " + МОДЕЛЬОБОРУДОВАНИЯ, ФР.ИМЯОБОРУДОВАНИЯ() );
RETURN false;
